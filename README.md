# Captura de prospectos 

## Información general

Api para ejecutar el proyecto

### Instrucciones de instalación

1. [Requisitos](Contar con node https://nodejs.org/es/download/)
2. [Requisitos](Se instala un servidor de base de datos como Xampp o WAMP para alojar la api)
3. [Instalación](se crea una base de datos vacía llamada "concredito" en PHPMyAdmin)
2. [Instalación](En la carpeta del proyecto instalar las librerias con el comando en consola de npm install)
3. [Configuración](Modificar el archivo ormconfig.json con las dependencias de base de datos)
4. [Ejecución](En la consola dentro de la carpeta del proyecto escribir npm run dev para ejecutar la api)
